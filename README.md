# Fast, async, multithreaded bomber

## Installation

```sh
git clone https://github.com/nezutero/tomaranai_bomber
```

## Usage

- Run it:

```sh
cd cmd
go run main.go
```

- Or run it using docker:

```sh
docker build -t your_image_name .
docker run -d -p 8080:80 your_image_name
```
